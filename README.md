# Lingo

Real-time AI translator. Text and speech. Web and mobile.

## Structure

```
lingo/
├── shared/     # Types, constants, design system tokens
├── web/        # React + Vite + TypeScript (web app)
└── mobile/     # React Native + Expo + TypeScript (mobile app)
```

## Getting started

```bash
# Install all workspaces
npm install

# Run web dev server
npm run web

# Run mobile (Expo)
npm run mobile
```
